Curso de Python

Shikamaru escolheria esse curso. Python é conhecida como uma linguagem com mais facilidade de aprendizado e ela consegue fazer coisas que outras linguagens fazem em apenas algumas linhas. Shikamaru, personagem conhecido como preguiçoso, sempre ver a forma mais facilitada para fazer algo escolheria o python.

Curso de Inteligência Artificial

A Ino se identificaria com esse curso. Por ter habilidades que envolva a mente onde é usado um ninjutsu de transferência de mente para poder coletar informações. Podemos fazer uma relação com a IA que pode ser usada para coleta e análise de informações semelhante ao ninjutsu da Ino.

Curso de HTML e CSS

Esse seria o curso perfeito para o personagem Sai e também entraria um curso de design. Ele usaria suas habilidades de desenhos para fazer telas e trazer elas a "vida".
